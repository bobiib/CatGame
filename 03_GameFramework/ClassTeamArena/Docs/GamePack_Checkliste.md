# GamePack-Checkliste (für Gruppen)

- [ ] Ordnername = PackId (ohne Leerzeichen)
- [ ] `story.md` + `rules.md` vorhanden (werden ins Output kopiert)
- [ ] >= 6 Actions (Keys eindeutig, A/B/C/D empfohlen)
- [ ] >= 6 Modules (Pressure/Event/Stabilisierung)
- [ ] Display-Labels passen zum Thema
- [ ] Win/Lose erklärt + Beispiel-Run in rules.md
